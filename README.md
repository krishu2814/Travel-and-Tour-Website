# Natours - Tour & Travel Website

A modern, responsive web application for a tour and travel business. Built with **Node.js**, **Express**, **MongoDB**, and **JavaScript**, this project allows users to explore and book tours online with smooth UI and secure backend features.

---

## Features

- View all available tours with detailed descriptions
- Book tours online
- User authentication and authorization
- Admin dashboard to manage tours, users, and bookings
- Email notifications for bookings and confirmations
- Responsive design for mobile and desktop
- Integration with MongoDB for data storage

---

## Tech Stack

- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Frontend:** HTML, CSS, JavaScript (or Pug/EJS if using templates)
- **Authentication:** JWT, bcrypt
- **Email Service:** NodeMailer
- **Others:** dotenv for environment variables, morgan for logging
